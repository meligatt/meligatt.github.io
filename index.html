Hola!

My name is Melissa, and I am Front-End developer with 8 years of experience in web development.

I have in-depth experience using modern javascript frameworks (with the last 4 being dedicated to React) to create user interfaces across a range of industries, including education, retail, and finance. I am proud to have advocated for, and implemented, unit and system testing with Jest and Cypress, as well as static type-checking with TypeScript, because I consider such practices fundamental to creating products that are scalable and maintainable.

I'm always putting effort into making the web available for everyone through best practices in web accessibility. Taking a pragmatic, incremental approach to implementing the W3C-WAI guidelines, I work to help businesses reach a larger pool of potential customers.

I have worked extensively with backend developers on RESTful and GraphQL API projects, discussing ideas and requirements for modeling the data with the client-side in mind. Having collaborated with Java, Ruby, and Python developers, I'm always keen to dig into backend languages to help with code design decisions.
